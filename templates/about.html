About the project

This application helps pharmacies manage inventories, orders, suppliers, and customer prescriptions. It’s built with Flask, SQLAlchemy, and Tailwind CSS for a clean, efficient, and responsive user experience.

Stack

Flask • SQLAlchemy • Tailwind CSS • JavaScript

Design

Soft blues, teals, and greens with smooth gradients and clean spacing.

Access

Secure role-based access (admin, pharmacist, doctor).
